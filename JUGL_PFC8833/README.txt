This is an open source driver supported by Jaycon Systems

Supported Screens: 
	Nokia Color LCD http://www.jayconsystems.com/color-lcd-shield-nokia-6100.html

	Supported Architectures:
		AVR

	Default Constructor:
		Not configurable, since this drives a shield with fixed pins

	Additional Details:
		Driver for PCF8833, wraps another driver written by Peter Barrett in 2010.
		This screen supports RGB color.
